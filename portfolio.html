Game Design

Cold Reception

C#, Unity

Cold Reception is the 1st game I developed utilizing 3D assets in order to develop my game design knowledge & skills and build my portfolio. This is is my 1st completely original game as well.

It's a 2.5D platformer-inspired game where you restart each level you die on. This is done to hook players by challenging them to get as far as they can without dying without forcing them to restart from level 1.

Volcic Breaker

C#, Unity

Volcic Breaker is a game I developed while taking a 2D game development course on Udemy in order to develop my game design knowledge & skills.

It's a 2D block-breaker game that uses common block-breaker concepts such as power-ups and accerlation physics. It also has a points system that aids you by letting you start over where you left off should you have enough points.

Lasik Infinite

C#, Unity

Lasik Infinite is a game I developed while taking a 2D game development course on Udemy in order to develop my game design knowledge & skills.

It's a top-down arcade-style shooter game that adapts the "Infinite Runner" formula to hook players into playing as long as possible.

Culture Quiz

C#, Unity

Culture Quiz is a game I developed while taking a 2D game development course on Udemy in order to develop my game design knowledge & skills.

It's a quiz game that uses focuses on simplicity over complex mechanics. The goal is to just act as a simple pass time activity like all those online quizzes people take.

App Development

Quiz Master

Android Studio, Java

Quiz Master is an app I developed while taking the "Mobile Programming" course at CSU - East Bay. The assignment was to creat a 2-player game app, and since this was the 1st game I was creating with an app dev environment and not a game engine like Unity, I chose to make a simple quiz game.

It's a quiz game that uses a points system and supports 2-4 players. The number of questions are proportional to the number of players.

Expense Tracking

Android Studio, Java

Expense Tracking is an app I developed while taking the "Mobile Programming" course at CSU - East Bay.

It's an app that allows you to enter expenses by category and date. It also supports adding notes to your expenses and has edit/delete/totals/averages funtionality.

Software Development

Boggle Simulator

Visual Studio, C++

"Boggle Simulator" is a program I finished developing as part of an assignment I did during my Data Structures & Algorithms course at CSU - East Bay.

The goal of the assignment was to simulate a game of boggle using a dictionary and a prefix-tree data strutcture. This would cut down the runtime by checking if any word with said prefix exists to avoid further searching.

Distributed File System

Eclipse, Java

The group project with 2 classmates in my software engineering course required us to develop a DFS program. The basis of the project was to code 1 server that could handle multiple clients and store their files privately or publicy.

I was responsible for coding the functionality of physically storing files and handling user identification, while my classmates handled coding the client-server side of things.

Video Production

PowerDirector

This is the 1st video I produced during my internship at Lawrence Berkeley National Lab. X-Ray Absorption & Fluorescence Spectroscopy

Adobe Premiere Pro

This is the 2nd video I produced during my internship at Lawrence Berkeley National Lab. Both of these videos were made with the notion of conveying a complex concept in a simplistic way so that general audiences will be able to understand.

This video was about 2 techniques used at the Advanced Light Source to analyze the inside structure of different materials. I chose to focus on techniques that dealt with x-rays because I felt that they'd be easier for general audiences to understand.
